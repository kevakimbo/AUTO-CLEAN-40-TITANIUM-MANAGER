# Auto Clean 40 - Pro Interface

Votre application de facturation "Pro" est prête.
Aucune installation n'est nécessaire.

## Comment l'utiliser ?
1. Ouvrez le dossier `auto_clean_40_pro`.
2. Double-cliquez sur le fichier `index.html`.
3. L'application s'ouvrira dans votre navigateur par défaut.

## Fonctionnalités
- **Tableau de bord** : Vue d'ensemble de votre chiffre d'affaires.
- **Éditeur** : Créez des Devis et Factures avec calcul automatique.
- **Historique** : Retrouvez tous vos documents sauvegardés (les données restent dans votre navigateur).
- **PDF** : Téléchargez vos documents en un clic.

## Personnalisation
Allez dans l'onglet **Paramètres** pour changer :
- Le nom de l'entreprise
- La couleur du thème (Jaune néon par défaut)
- Vos services rapides
